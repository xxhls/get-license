# GET-LICENSE

[en](./README.md) | [简体中文](./README-zh.md)

## Introduction

A tool to help you generate license files for your project

## Guide

### Setup

```shell
npm install @xxhls/get-license -g
```

### Useage

```shell
get-license --license=mit
```
