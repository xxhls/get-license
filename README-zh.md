# GET-LICENSE

[en](./README.md) | [简体中文](./README-zh.md)

## 简介

帮助您为项目生成许可证文件的工具

## 指引

### 安装

```shell
npm install @xxhls/get-license -g
```

### 使用

```shell
get-license --license=mit
```
